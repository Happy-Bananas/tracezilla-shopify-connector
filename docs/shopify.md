# Shopify

Before we can synchronize products from Shopify into Tracezilla, we need a safe environment for developing and testing our integration.

In this section, we will set up everything required to build, test, and debug the connector locally. When you’re finished, you’ll have a fully functional Shopify environment that you can use with this example project or your own applications.

No prior Shopify experience is required. We’ll go through each step together and explain why it is needed.

⸻

What You Will Build

By the end of this section, you will have:

* ✅ A Shopify Partner account
* ✅ A Shopify Development Store
* ✅ A Shopify Custom App
* ✅ API credentials for local development
* ✅ A store that you can populate with test products

Everything in this section can be created and tested without affecting a production store or exposing real products for sale.
