# Build a Shopify to Tracezilla Integration

Learn how to connect Shopify and Tracezilla using Laravel, step by step.

## What is this?

The Tracezilla Shopify Connector is an open-source Laravel application that demonstrates how to synchronize product catalogs from Shopify into Tracezilla.

It is both a working example and a starting point for your own integrations.

Whether you're new to Shopify, Tracezilla, or Laravel, the guides in this documentation will help you get up and running quickly.

## Who is this for?

This documentation is for you if:

- You need to integrate Shopify with Tracezilla.
- You want a practical Laravel example.
- You prefer learning by running real code.
- You want a foundation for your own integration.

## What will I learn?

By the end of this guide, you will know how to:

- Run the project locally.
- Create Shopify development credentials.
- Connect to Tracezilla.
- Synchronize your first product catalog.
- Extend the integration for your own use case.

## What do I need?

Prerequisites:

- Git
- Docker (recommended but not required)
- A Shopify Partner account
- A Tracezilla account

Docker is recommended to make getting started easy, but it is not a requirement.

## Where should I start?

1. Installation
2. Create a Shopify development store
3. Create a Shopify custom app
4. Configure Tracezilla credentials
5. Run your first synchronization
6. Explore the architecture


May your syncs always be successful. 🚀
