# tracezilla


In this section, you will confugure everything required to connect the [Shopify larval tracezilla](https://github.com/Happy-Bananas/tracezilla-shopify-connector) demo application to the tracezilla API. When you’re finished, you’ll have a fully configured tracezilla demo environment that you can use with this example project or your own integrations.

No prior tracezilla experience is required. We’ll go through each step together and explain why it is needed.

>⚠️ Notice <br/>
A tracezilla demo account expires after 14 days. If you need more time, you can [contact us](https://www.tracezilla.com/en/contact-us) to request an extension.

---

## What You Will Build

By the end of this section, you will have:

* ✅ A tracezilla demo account
* ✅ A tracezilla organization
* ✅ A tracezilla API integration


Everything in this section can be created and tested without affecting production data. The credentials you create will only be used by the example application to authenticate with the tracezilla API.
